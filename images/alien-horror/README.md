## Source

AI-generated wallpaper set inspired by the evolving production design of the
classic *Alien* film series. The scenes progress from analog deep-space horror,
through a storm-battered industrial colony, to a derelict station beside a
planetary ring.

The set emphasizes empty environments, practical machinery, low-key lighting,
and implied threats. It does not use actor likenesses, readable logos, or film
stills.

### Hourly sequence

- `0-7`: retro analog spacecraft and biomechanical ruins
- `8-15`: colonial military-industrial environments
- `16-23`: mining colony, derelict station, and planetary rings
